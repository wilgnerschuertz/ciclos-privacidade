Política de Privacidade — Ciclos+
Última atualização: janeiro de 2026
O Ciclos+ foi desenvolvido com foco em simplicidade, transparência e respeito à privacidade. Esta Política de Privacidade explica como as informações são tratadas no aplicativo.
1. Coleta de informações
O Ciclos+ não coleta, armazena ou compartilha dados pessoais do usuário em servidores externos.
Todas as informações inseridas no aplicativo, como dados do ciclo menstrual e preferências de lembretes, permanecem exclusivamente no dispositivo do usuário.
2. Uso das informações
Os dados inseridos no Ciclos+ são utilizados apenas para o funcionamento do aplicativo, como:
Exibição do calendário do ciclo
Cálculo e visualização de períodos
Agendamento de lembretes, quando ativados pelo usuário
Essas informações não são utilizadas para fins comerciais, publicitários ou analíticos.
3. Notificações
O Ciclos+ pode solicitar permissão para enviar notificações com o objetivo de:
Lembrar o usuário de registrar informações diárias
Enviar lembretes de hidratação, caso essa opção seja ativada
As notificações são opcionais e podem ser ativadas ou desativadas a qualquer momento nas configurações do aplicativo ou do sistema.
4. Compartilhamento de dados
O Ciclos+ não compartilha dados com terceiros.
Não há integração com redes sociais, serviços de publicidade ou plataformas de rastreamento.
5. Compras no aplicativo
O Ciclos+ oferece uma opção opcional de apoio financeiro por meio de compra dentro do aplicativo.
Nenhum dado pessoal adicional é coletado durante esse processo. As transações são processadas diretamente pela Apple, conforme suas próprias políticas de privacidade.
6. Segurança
Como os dados permanecem localmente no dispositivo do usuário, a segurança das informações depende das proteções do próprio sistema operacional (iOS), como bloqueio por senha, Face ID ou Touch ID.
7. Alterações nesta política
Esta Política de Privacidade pode ser atualizada ocasionalmente para refletir melhorias no aplicativo ou ajustes legais. Qualquer alteração será comunicada de forma clara.
